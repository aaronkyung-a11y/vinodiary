# Vino·Diary CHANGELOG

> **v0.26 ~ v1.18 항목은 사후 복원분입니다.** 이 구간은 개발 당시 CHANGELOG에
> 기록되지 않아, 각 버전 커밋 스냅샷의 코드 diff를 근거로 되살린 내용입니다.
> 즉 코드 변화에서 **추정**한 요약이라 실제 의도와 세부가 다를 수 있고, 커밋되지
> 않고 건너뛴 버전 번호(예: v0.23~v0.25, v0.33, v0.46, v0.51~v0.52 등)는
> 실제 배포되지 않아 목록에 없습니다. "백엔드: 새 action" 표기는 프론트 코드에서
> 새 `action` 호출이 등장한 시점을 근거로 표시했습니다.
